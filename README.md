# HCI_Projet

## Questions pour nos interviews
1) A combien de manifestations êtes-vous allé cette année?
2) Pourquoi manifestez-vous?
3) Avez-vous déjà organisé une manifestation?
4) Quels problèmes avez-vous eu?
5) Comment pouvez-vous gerer le nombre de personnes?
6) Comment disséminez vous l'information?
7) Quels problèmes avez-vous eu en tant que participant?
8) Est-ce qu'il y a une manif en particulier qui vous a marqué (positivement ou négativement) et pourquoi?
9) Comment trouvez-vous les informations?
10) Selon vous, qu'est-ce qui détermine qu'une manif est réussie?
